# Приложение для управления задачами (PWA)

## Описание проекта

Веб-приложение для управления списком задач с поддержкой:

- Работы без сети (service workers)
- Push-уведомлений
- Напоминаний с возможностью их отложить
- Установки на устройство (PWA)
